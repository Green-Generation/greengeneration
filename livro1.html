🧠 O que é Hardware?

Hardware é a parte física do computador – tudo o que você pode tocar: placas, cabos, periféricos, fontes, etc. Ele trabalha junto com o software (os programas e sistemas que rodam na máquina).

🖥️ Oque são hardwares de: Entrada, Saída, Entrada/Saída e Processamento

1- Hardware de Entrada

São os dispositivos usados para enviar informações para o computador.

2- Hardware de Saída

São os componentes responsáveis por mostrar as informações do computador para o usuário.

3- Hardware de Entrada e Saída

Alguns dispositivos servem tanto para enviar quanto para receber informações do computador.

4- Hardware de Processamento

São os responsáveis por realizar os cálculos e operações lógicas.

🧩 Componentes Principais do Computador

🔲 Placa-mãe

Conecta todos os componentes e distribui energia entre eles. Tem entradas para CPU, RAM, armazenamento, etc.

⚙️ Processador

Executa tarefas e cálculos. Ex: Intel Core i5, AMD Ryzen 7. Medido em GHz.

💾 Memória RAM

Armazena dados temporariamente durante o uso. Mais RAM = mais desempenho multitarefa.

📀 Armazenamento

Guarda dados permanentemente. Tipos:

  • HDD: Barato e espaçoso
  • SSD: Rápido e confiável
  • NVMe: Extremamente rápido

🎮 Placa de Vídeo

Processa gráficos e melhora desempenho em jogos, edições, etc.

🔌 Fonte de Alimentação

Distribui energia para os componentes. Medida em watts (W).

🧊 Resfriamento

Evita superaquecimento. Inclui coolers, dissipadores, ou sistemas líquidos.

🖥️ Gabinete

O gabinete é aonde voce colocas as peças de hardware e la fica guardado os componentes.

💻 Diferença entre Hardware e Software

O hardware é a parte física do computador, tudo aquilo que podemos tocar. São os componentes como o monitor, o teclado, o mouse, o HD, a placa-mãe, a memória RAM e o processador.

O software são os programas e sistemas que fazem o computador funcionar. Exemplos o sistemaoperacional o Windows ou Linux, e os proprios aplicativos que baixamos no computador tipo navegadores de internet,jogos e aplicativos entre varios outros

🛠️ Manutenção Básica de Hardware

⚡ 1- Desligar corretamente:

Sempre desligue o computador pelo sistema clicando na opção de desligar e evite tirar o cabo da tomada diretamente.

🧹 2- Limpeza externa e interna
Com o tempo, o computador acumula poeira, o que pode causar superaquecimento. Então:

• Limpe sempre a parte externa com um pano seco ou levemente úmido

• Se tiver um pouco de experiência e confiança, abra o gabinete com cuidado e use um pincel macio para remover a poeira.

🌡️ 3- Evite o superaquecimento

Coloque o computador em um local com boa ventilação e que não bloqueie as saídas de ar, com isso podemos evitar o superaquecimento

🔌 4- Verifique os cabos e conexões

Confira se os cabos estão bem conectados e em bom estado.

🔧 5- Assistência técnica

Sempre que precisar arrumar o computador com algo que voce não tenha muito experiencia, lembre-se de sempre levar para especialista que saibam mecher com computador, para você não ter nenhum tipo de problema pois é bem delicado o computador qualquer erro básico voce pode ter um prejuizo grande.


📚 Glossário de Hardware

🔌 Placa-mãe: A parte principal do computador onde conectamos todos os cabos do PC.

🧠 CPU: É o processador do computador, responsável por executar tarefas e cálculos. Ele é considerado o cerébro do computador.

💾 Memória RAM: É onde os dados são armazenados temporariamente.

📀 HD: É o disco rígido usado para armazenar arquivos e programas de forma permanente.

💨 Cooler: É a ventoinha do computador que ajuda a manter uma temperatura adequada.

🎮 PCI Express: É o slot onde conectamos as placas de vídeo.

⚙️ BIOS: Inicializa e configura os componentes de hardware, como processador e memória RAM.

🔋 Fonte de alimentação: Responsável por distribuir energia para o computador.

🖥️ GPU: É a placa de vídeo, responsável por processar imagens e vídeos.
